## 引用

```scss
// 在指定的文件中或 app.scss 中引入对应样式 text.scss
@import '~miao-ui/styles/cores/text.scss' // 如不使用框架则建议这个
// 或
@import '~miao-ui/styles/index.scss'
```

## 示例

```ts
<Text className="text-">
    测试文字
</Text>
```
